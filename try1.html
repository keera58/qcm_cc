ABOUT PROJECT

This Project Calculates the Feedback given by students
Steps Description
Click on Polling The user needs to click on the Tab "Polling" given in the navigation bar.
Select the Courses Then the user needs to select the course for which he wants to give the feedback.
Select the Faculty Then the user needs to select the Faculty only after selecting for whic he/she wants to give feedback and ratings.
Give rating Then the user needs to give star ratings on a scale of 1 to 10 based on his choice
Submit The user needs to click on the submit button only after giving ratings

*All Fields are mandatory
